# Using it on desktop

On desktop the full screen width is available – the map and detail view are
visible side by side.

![Full view on desktop](/assets/screenshots/user/desktop-overview.png)

## Operating the map

- **Pan:** drag with the mouse.
- **Zoom:** the mouse wheel or the +/− buttons.
- **Open a tent:** click a marker.

## Search & filter

The search bar and category buttons are permanently visible at the top – no
expanding needed. See [Selecting a tent](/en/user/select-tent).

## Detail side panel

A selected tent appears as a **side panel on the right** while the map stays
visible. This makes it easy to switch between tents.

## Viewing photos

Photos open as a lightbox over the page:

- **Browse:** arrow keys ← →
- **Close:** Esc, a click on the background, or the ✕

## Keyboard

The detail view holds keyboard focus so content can be navigated with Tab –
helpful for accessibility.
