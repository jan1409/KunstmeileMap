# Selecting a tent

There are three ways to find a tent: via the **map**, via **search**, or via the
**category filter**.

## Via the map

Just tap a [marker](/en/user/map-basics#understanding-markers). The map brings
the tent into focus and opens the detail view.

## Via search

The search bar finds tents by **name** or **contact person** – as you type:

![Search by name or contact person](/assets/screenshots/user/search-dropdown.png)

Tap a result in the list to jump straight to the tent.

## Via the category filter

The category buttons let you narrow the map to certain kinds of tents:

![Category filter](/assets/screenshots/user/category-filter.png)

- Tap one or more categories to show only those tents.
- **All** clears the filter again.

## The detail view

Once a tent is selected, a detail view opens with all the info:

![Tent detail side panel](/assets/screenshots/user/side-panel.png)

Depending on what's been filled in, it shows the name and number, description,
contact person, categories, address, links (website, Instagram, Facebook) and
photos.

::: info Layout
On desktop the detail view appears as a **side panel on the right**, on mobile as
a **drawer from the bottom**. More on this under [Mobile](/en/user/mobile) and
[Desktop](/en/user/desktop).
:::

## Viewing photos (lightbox)

A click/tap on a photo opens the large view (lightbox):

![Photo lightbox](/assets/screenshots/user/photo-lightbox.png)

- **Browse:** arrow keys ← → (desktop) or swipe sideways (mobile).
- **Close:** the Esc key, a click on the background, or the ✕.
